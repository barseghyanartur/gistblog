Hide *.py,cover files from IDE

2026-04-02 | Tech | python, vs-code, pycharm

VSCode

Settings -> Text Editor -> Files -> Add Pattern: Add **/*.py,cover

PyCharm

Editor -> File Types -> Ignored Files and Folders : Add *.py,cover
